House of Cliche Horrors:

This is a choose-your-own adventure game written in JavaScript and styled using CSS3 and BootStrap. Players must survive in the haunted mansion until the timer counts down to zero and opens the front door! But beware! For monsters lurk around every corner!

Past-Tents

This is a social media campsite review site wherein users can log in and add their favorite campgrounds and add reviews to other campgrounds. This application was built using MongoDB, HTML5, CSS3, and BootStrap.

(g)HOST/Rider

This is a website wherein user can log in and post their vehicles to be rented or to rent other users' vehicles. This application was built using Django, SQLite, and React.
